---
title: Selling Partner API for Uploads
description: "The Uploads API lets you upload files that you can programmatically access using other Selling Partner APIs, such as the A+ Content API and the Messaging API."
---

# API List
- [createUploadDestinationForResource](operations/createUploadDestinationForResource.json) (POST /uploads/2020-11-01/uploadDestinations/{resource})：Creates an upload destination, returning the information required to upload a file to the destination and to programmatically access the file.
