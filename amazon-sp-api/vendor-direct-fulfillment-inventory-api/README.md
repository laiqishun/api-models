# API List
- [submitInventoryUpdate](operations/submitInventoryUpdate.json) (POST /vendor/directFulfillment/inventory/v1/warehouses/{warehouseId}/items)：Submits inventory updates for the specified warehouse for either a partial or full feed of inventory items.
