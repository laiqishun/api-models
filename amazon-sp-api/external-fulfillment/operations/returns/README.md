description: "You can use the Amazon External Fulfillment Return Item Processing API to retrieve, track, and process return items through Amazon's External Fulfillment returns management system."
---

# API List
- [listReturns](listReturns.json) (GET /externalFulfillment/2024-09-11/returns)：Retrieve a list of return items. You can filter results by location, RMA ID, status, or time.
- [getReturn](getReturn.json) (GET /externalFulfillment/2024-09-11/returns/{returnId})：Retrieve the return item with the specified ID.
