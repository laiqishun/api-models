# SP-API Offline Reference Values

This folder contains short local reference pages for SP-API fields whose valid values are normally described only in Amazon's online documentation.

Use these files when an operation description says to refer to one of these value pages:

| Online value page | Local offline page | Common fields |
| --- | --- | --- |
| Report Type Values | [report-type-values.md](report-type-values.md) | `reportType`, `reportTypes` |
| Feed Type Values | [feed-type-values.md](feed-type-values.md) | `feedType`, `feedTypes` |
| Notification Type Values | [notification-type-values.md](notification-type-values.md) | `notificationType`, `eventFilterType`, `filterExpression` |
| Marketplace IDs / Store Identifiers | [marketplace-ids.md](marketplace-ids.md) | `marketplaceId`, `marketplaceIds` |

These files are intentionally concise. They are not full mirrors of the online docs. Prefer the exact uppercase strings shown here when filling request parameters.
