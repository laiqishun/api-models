# API List
- [SubmitShipmentConfirmations](operations/SubmitShipmentConfirmations.json) (POST /vendor/shipping/v1/shipmentConfirmations)：Submits one or more shipment confirmations for vendor orders.
- [SubmitShipments](operations/SubmitShipments.json) (POST /vendor/shipping/v1/shipments)：Submits one or more shipment request for vendor Orders.
- [GetShipmentDetails](operations/GetShipmentDetails.json) (GET /vendor/shipping/v1/shipments)：Returns the Details about Shipment, Carrier Details, status of the shipment, container details and other details related to shipment based on the filter parameters value that you specify.
- [GetShipmentLabels](operations/GetShipmentLabels.json) (GET /vendor/shipping/v1/transportLabels)：Returns small parcel shipment labels based on the filters that you specify.
