# API List
- [searchCatalogItems](operations/searchCatalogItems.json) (GET /catalog/2022-04-01/items)：Search for a list of Amazon catalog items and item-related information. You can search by identifier or by keywords.
- [getCatalogItem](operations/getCatalogItem.json) (GET /catalog/2022-04-01/items/{asin})：Retrieves details for an item in the Amazon catalog.
