title: The Selling Partner API for FBA inbound operations.
description: "The Selling Partner API for Fulfillment By Amazon (FBA) Inbound. The FBA Inbound API enables building inbound workflows to create, manage, and send shipments into Amazon's fulfillment network. The API has interoperability with the Send-to-Amazon user interface."
---

# API List
- [listInboundPlans](operations/listInboundPlans.json) (GET /inbound/fba/2024-03-20/inboundPlans)：Provides a list of inbound plans with minimal information.
- [createInboundPlan](operations/createInboundPlan.json) (POST /inbound/fba/2024-03-20/inboundPlans)：Creates an inbound plan. An inbound plan contains all the necessary information to send shipments into Amazon's fufillment network.
- [getInboundPlan](operations/getInboundPlan.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId})：Fetches the top level information about an inbound plan.
- [listInboundPlanBoxes](operations/listInboundPlanBoxes.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/boxes)：Provides a paginated list of box packages in an inbound plan.
- [cancelInboundPlan](operations/cancelInboundPlan.json) (PUT /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/cancellation)：Cancels an Inbound Plan. Charges may apply if the cancellation is performed outside of a void window. The window for Amazon Partnered Carriers is 24 hours for Small Parcel Delivery (SPD) and one hour for Less-Than-Truckload (LTL) carrier shipments.
- [listInboundPlanItems](operations/listInboundPlanItems.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/items)：Provides a paginated list of item packages in an inbound plan.
- [updateInboundPlanName](operations/updateInboundPlanName.json) (PUT /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/name)：Updates the name of an existing inbound plan.
- [listPackingGroupBoxes](operations/listPackingGroupBoxes.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/packingGroups/{packingGroupId}/boxes)：Retrieves a page of boxes from a given packing group. These boxes were previously provided through the `setPackingInformation` operation. This API is used for workflows where boxes are packed before Amazon determines shipment splits.
- [listPackingGroupItems](operations/listPackingGroupItems.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/packingGroups/{packingGroupId}/items)：Retrieves a page of items in a given packing group. Packing options must first be generated by the corresponding operation before packing group items can be listed.
- [setPackingInformation](operations/setPackingInformation.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/packingInformation)：Sets packing information for an inbound plan. This should be called after an inbound plan is created to populate the box level information required for planning and transportation estimates.
- [listPackingOptions](operations/listPackingOptions.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/packingOptions)：Retrieves a list of all packing options for an inbound plan. Packing options must first be generated by the corresponding operation before becoming available.
- [generatePackingOptions](operations/generatePackingOptions.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/packingOptions)：Generates available packing options for the inbound plan.
- [confirmPackingOption](operations/confirmPackingOption.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/packingOptions/{packingOptionId}/confirmation)：Confirms the packing option for an inbound plan.
- [listInboundPlanPallets](operations/listInboundPlanPallets.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/pallets)：Provides a paginated list of pallet packages in an inbound plan. An inbound plan will have pallets when the related details are provided after generating Less-Than-Truckload (LTL) carrier shipments.
- [listPlacementOptions](operations/listPlacementOptions.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/placementOptions)：Provides a list of all placement options for an inbound plan. Placement options must first be generated by the corresponding operation before becoming available.
- [generatePlacementOptions](operations/generatePlacementOptions.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/placementOptions)：Generates placement options for the inbound plan.
- [confirmPlacementOption](operations/confirmPlacementOption.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/placementOptions/{placementOptionId}/confirmation)：Confirms the placement option for an inbound plan. Once confirmed, it cannot be changed for the Inbound Plan.
- [getShipment](operations/getShipment.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId})：Provides the full details for a specific shipment within an inbound plan. The `transportationOptionId` inside `acceptedTransportationSelection` can be used to retrieve the transportation details for the shipment.
- [listShipmentBoxes](operations/listShipmentBoxes.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/boxes)：Provides a paginated list of box packages in a shipment.
- [listShipmentContentUpdatePreviews](operations/listShipmentContentUpdatePreviews.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/contentUpdatePreviews)：Retrieve a paginated list of shipment content update previews for a given shipment. The shipment content update preview is a summary of the requested shipment content changes along with the transportation cost implications of the change that can only be confirmed prior to the expiry date specified.
- [generateShipmentContentUpdatePreviews](operations/generateShipmentContentUpdatePreviews.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/contentUpdatePreviews)：Generate a shipment content update preview given a set of intended boxes and/or items for a shipment with a confirmed carrier. The shipment content update preview will be viewable with the updated costs and contents prior to confirmation.
- [getShipmentContentUpdatePreview](operations/getShipmentContentUpdatePreview.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/contentUpdatePreviews/{contentUpdatePreviewId})：Retrieve a shipment content update preview which provides a summary of the requested shipment content changes along with the transportation cost implications of the change that can only be confirmed prior to the expiry date specified.
- [confirmShipmentContentUpdatePreview](operations/confirmShipmentContentUpdatePreview.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/contentUpdatePreviews/{contentUpdatePreviewId}/confirmation)：Confirm a shipment content update preview and accept the changes in transportation cost.
- [getDeliveryChallanDocument](operations/getDeliveryChallanDocument.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/deliveryChallanDocument)：Provide delivery challan document for PCP transportation in IN marketplace.
- [listDeliveryWindowOptions](operations/listDeliveryWindowOptions.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/deliveryWindowOptions)：Retrieves all delivery window options for a shipment. Delivery window options must first be generated by the `generateDeliveryWindowOptions` operation before becoming available.
- [generateDeliveryWindowOptions](operations/generateDeliveryWindowOptions.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/deliveryWindowOptions)：Generates available delivery window options for a given shipment.
- [confirmDeliveryWindowOptions](operations/confirmDeliveryWindowOptions.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/deliveryWindowOptions/{deliveryWindowOptionId}/confirmation)：Confirms the delivery window option for chosen shipment within an inbound plan. A placement option must be confirmed prior to use of this API. Once confirmed, new delivery window options cannot be generated, but the chosen delivery window option can be updated before shipment closure. The window is used to provide the expected time when a shipment will arrive at the warehouse. All transportation options which have the program `CONFIRMED_DELIVERY_WINDOW` require a delivery window to be confirmed prior to transportation option confirmation.
- [listShipmentItems](operations/listShipmentItems.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/items)：Provides a paginated list of item packages in a shipment.
- [updateShipmentName](operations/updateShipmentName.json) (PUT /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/name)：Updates the name of an existing shipment.
- [listShipmentPallets](operations/listShipmentPallets.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/pallets)：Provides a paginated list of pallet packages in a shipment. A palletized shipment will have pallets when the related details are provided after generating Less-Than-Truckload (LTL) carrier shipments.
- [cancelSelfShipAppointment](operations/cancelSelfShipAppointment.json) (PUT /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/selfShipAppointmentCancellation)：Cancels a self-ship appointment slot against a shipment. Only available in the following [marketplaces](https://developer-docs.amazon.com/sp-api/docs/marketplace-ids): MX, BR, EG, SA, AE, IN.
- [getSelfShipAppointmentSlots](operations/getSelfShipAppointmentSlots.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/selfShipAppointmentSlots)：Retrieves a list of available self-ship appointment slots used to drop off a shipment at a warehouse. Only available in the following [marketplaces](https://developer-docs.amazon.com/sp-api/docs/marketplace-ids): MX, BR, EG, SA, AE, IN.
- [generateSelfShipAppointmentSlots](operations/generateSelfShipAppointmentSlots.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/selfShipAppointmentSlots)：Initiates the process of generating the appointment slots list. Only available in the following [marketplaces](https://developer-docs.amazon.com/sp-api/docs/marketplace-ids): MX, BR, EG, SA, AE, IN.
- [scheduleSelfShipAppointment](operations/scheduleSelfShipAppointment.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/selfShipAppointmentSlots/{slotId}/schedule)：Confirms or reschedules a self-ship appointment slot against a shipment. Only available in the following [marketplaces](https://developer-docs.amazon.com/sp-api/docs/marketplace-ids): MX, BR, EG, SA, AE, IN.
- [updateShipmentSourceAddress](operations/updateShipmentSourceAddress.json) (PUT /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/sourceAddress)：Updates the source address of an existing shipment. The shipment source address can only be updated prior to the confirmation of the shipment carriers. As a result of the updated source address, existing transportation options will be invalidated and will need to be regenerated to capture the potential difference in transportation options and quotes due to the new source address.
- [updateShipmentTrackingDetails](operations/updateShipmentTrackingDetails.json) (PUT /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/shipments/{shipmentId}/trackingDetails)：Updates a shipment's tracking details.
- [listTransportationOptions](operations/listTransportationOptions.json) (GET /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/transportationOptions)：Retrieves all transportation options for a shipment. Transportation options must first be generated by the `generateTransportationOptions` operation before becoming available.
- [generateTransportationOptions](operations/generateTransportationOptions.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/transportationOptions)：Generates available transportation options for a given placement option.
- [confirmTransportationOptions](operations/confirmTransportationOptions.json) (POST /inbound/fba/2024-03-20/inboundPlans/{inboundPlanId}/transportationOptions/confirmation)：Confirms all the transportation options for an inbound plan. A placement option must be confirmed prior to use of this API. Once confirmed, new transportation options can not be generated or confirmed for the Inbound Plan.
- [listItemComplianceDetails](operations/listItemComplianceDetails.json) (GET /inbound/fba/2024-03-20/items/compliance)：List the inbound compliance details for MSKUs in a given marketplace.
- [updateItemComplianceDetails](operations/updateItemComplianceDetails.json) (PUT /inbound/fba/2024-03-20/items/compliance)：Update compliance details for a list of MSKUs. The details provided here are only used for the India (IN - A21TJRUUN4KGV) marketplace compliance validation.
- [createMarketplaceItemLabels](operations/createMarketplaceItemLabels.json) (POST /inbound/fba/2024-03-20/items/labels)：For a given marketplace - creates labels for a list of MSKUs.
- [listPrepDetails](operations/listPrepDetails.json) (GET /inbound/fba/2024-03-20/items/prepDetails)：Get preparation details for a list of MSKUs in a specified marketplace.\n\n**Note:** MSKUs that contain certain characters must be encoded. For more information, refer to [URL Encoding](https://developer-docs.amazon.com/sp-api/docs/url-encoding).\n\nThe following characters must be double percent encoded:\n\n- `%`\n- `+`\n- `,`\n\n**Examples:** An MSKU value of `test%msku` is encoded as `test%2525msku`. An MSKU value of `test,msku` is encoded as `test%252Cmsku`.
- [setPrepDetails](operations/setPrepDetails.json) (POST /inbound/fba/2024-03-20/items/prepDetails)：Set the preparation details for a list of MSKUs in a specified marketplace.
- [getInboundOperationStatus](operations/getInboundOperationStatus.json) (GET /inbound/fba/2024-03-20/operations/{operationId})：Gets the status of the processing of an asynchronous API call.
