# API List
- [getMarketplaceParticipations](operations/getMarketplaceParticipations.json) (GET /sellers/v1/marketplaceParticipations)：Returns a list of marketplaces where the seller can list items and information about the seller's participation in those marketplaces.
- [getAccount](operations/getAccount.json) (GET /sellers/v1/account)：Returns information about a seller account and its marketplaces.
