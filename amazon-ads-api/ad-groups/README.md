# API List
- [CreateAdGroup](operations/CreateAdGroup.json) (POST /adsApi/v1/create/adGroups)：Create ad groups
- [DeleteAdGroup](operations/DeleteAdGroup.json) (POST /adsApi/v1/delete/adGroups)：Delete ad groups
- [QueryAdGroup](operations/QueryAdGroup.json) (POST /adsApi/v1/query/adGroups)：List ad groups
- [UpdateAdGroup](operations/UpdateAdGroup.json) (POST /adsApi/v1/update/adGroups)：Update ad groups
