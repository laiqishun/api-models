# API List
- [CreateGeoLocation](operations/CreateGeoLocation.json) (POST /adsApi/v1/create/geoLocations)：Create geo location targeting definitions. Supports smart locations, which target users based on their percentile rank within a Smart Location Index, and radius locations, which target users within a specified distance of an address or coordinate. Note: radius locations are currently in beta.
