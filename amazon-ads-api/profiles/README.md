---
title: Profiles
description: "Profiles represent an advertiser and their account's marketplace, and are used in all subsequent API calls via a management scope, `Amazon-Advertising-API-Scope`."
---

# API List
- [getProfileById](operations/getProfileById.json) (GET /v2/profiles/{profileId})：Gets a profile specified by identifier.
- [listProfiles](operations/listProfiles.json) (GET /v2/profiles)：Gets a list of profiles.
- [updateProfiles](operations/updateProfiles.json) (PUT /v2/profiles)：Update the daily budget for one or more profiles.
