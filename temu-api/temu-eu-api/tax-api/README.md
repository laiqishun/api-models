# Tax API

Source group: `5、Tax`.

## 使用范围

欧洲税务、VAT 与相关税务信息（tax, VAT）。

先按上级区域 README 确定店铺模式和网关，再从下方列表选择唯一 operation。

## API List

- [temu.pay.tax.invoice.info.query](operations/temu.pay.tax.invoice.info.query.json) - Support batch query of invoice information associated with order numbers.
- [temu.pay.tax.invoice.pdf.download](operations/temu.pay.tax.invoice.pdf.download.json) - Support downloading specified invoice PDF file according to invoice number.
