# Promotion API

Source group: `7、Promotion`.

## API List

- [bg.promotion.activity.candidate.goods.query](operations/bg.promotion.activity.candidate.goods.query.json) - the local to local activity candidate goods
- [bg.promotion.activity.goods.enroll](operations/bg.promotion.activity.goods.enroll.json) - enroll products in the local to local activity
- [bg.promotion.activity.goods.operation.query](operations/bg.promotion.activity.goods.operation.query.json) - query the result of operation in the local to local activity
- [bg.promotion.activity.goods.query](operations/bg.promotion.activity.goods.query.json) - query the local to local activity goods
- [bg.promotion.activity.goods.update](operations/bg.promotion.activity.goods.update.json) - update activity goods information in the local to local activity
- [bg.promotion.activity.query](operations/bg.promotion.activity.query.json) - query the local to local activity
