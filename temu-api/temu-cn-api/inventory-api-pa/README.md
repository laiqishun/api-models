# Inventory PA API

Source group: `库存管理API组-PA`.

## API List

- [bg.btg.goods.stock.quantity.get](operations/bg.btg.goods.stock.quantity.get.json) - OpenApi查询半托管商品销售库存
- [bg.btg.goods.stock.quantity.update](operations/bg.btg.goods.stock.quantity.update.json) - OpenApi更新半托管商品库存
- [bg.btg.goods.stock.route.add](operations/bg.btg.goods.stock.route.add.json) - OpenApi新增半托管商品仓站关系和库存
- [bg.btg.goods.stock.warehouse.list.get](operations/bg.btg.goods.stock.warehouse.list.get.json) - 根据站点列表查询自发货模式品可绑定的发货仓信息
