# Video Upload API

Source group: `视频上传API组`.

## 使用范围

视频上传签名与转码结果查询（video upload, transcoding）。

先按上级区域 README 确定店铺模式和网关，再从下方列表选择唯一 operation。

## API List

- [bg.goods.big.video.upload.result.get.global](operations/bg.goods.big.video.upload.result.get.global.json) - 查询视频转码结果接口
- [bg.goods.video.upload.sign.get.global](operations/bg.goods.video.upload.sign.get.global.json) - 视频文件上传sign查询
