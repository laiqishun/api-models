# Model Fitting API

Source group: `商品模特试穿API组`.

## 使用范围

模特资料、适用类目与试穿信息（model, fitting）。

先按上级区域 README 确定店铺模式和网关，再从下方列表选择唯一 operation。

## API List

- [bg.modelcats.get](operations/bg.modelcats.get.json) - 可添加模特类目查询
- [bg.modelinfo.add](operations/bg.modelinfo.add.json) - 新增模特信息
- [bg.modelinfo.edit](operations/bg.modelinfo.edit.json) - 编辑模特信息
- [bg.modelinfo.get](operations/bg.modelinfo.get.json) - 模特信息查询
