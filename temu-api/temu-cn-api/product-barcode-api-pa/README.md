# Product Barcode PA API

Source group: `商品条码API组-PA`.

## 使用范围

Partner 网关商品条码与标签查询（barcode, label）。

先按上级区域 README 确定店铺模式和网关，再从下方列表选择唯一 operation。

## API List

- [bg.glo.goods.custom.label.get](operations/bg.glo.goods.custom.label.get.json) - 提供查询定制品条码信息接口，用于条码打印
- [bg.glo.goods.labelv2.get](operations/bg.glo.goods.labelv2.get.json) - 提供查询商品条码信息接口，用于条码打印
