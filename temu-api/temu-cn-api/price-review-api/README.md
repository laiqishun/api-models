# Price Review API

Source group: `申报价/核价/调价API`.

## API List

- [bg.full.adjust.price.batch.review](operations/bg.full.adjust.price.batch.review.json) - 支持大卖自研商家批量操作调价单
- [bg.full.adjust.price.page.query](operations/bg.full.adjust.price.page.query.json) - 支持大卖自研商家查询调价单
- [bg.price.review.confirm](operations/bg.price.review.confirm.json) - 同意核价单建议价
- [bg.price.review.page.query](operations/bg.price.review.page.query.json) - 支持商家查询核价单
- [bg.price.review.reject](operations/bg.price.review.reject.json) - 支持商家不同意全托管核价单建议价（并给出新的申报价）
