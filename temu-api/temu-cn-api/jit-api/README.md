# JIT API

Source group: `JIT组`.

## 使用范围

全托 JIT 模式、虚拟库存与规则（JIT, virtual inventory）。

先按上级区域 README 确定店铺模式和网关，再从下方列表选择唯一 operation。

## API List

- [bg.virtualinventoryjit.edit](operations/bg.virtualinventoryjit.edit.json) - 用于更新jit商品的库存
- [bg.virtualinventoryjit.get](operations/bg.virtualinventoryjit.get.json) - 用于jit商品的库存查询
- [bg.virtualinventoryjit.rule.get](operations/bg.virtualinventoryjit.rule.get.json) - jit预售规则查询接口
- [bg.virtualinventoryjit.rule.sign](operations/bg.virtualinventoryjit.rule.sign.json) - jit预售规则签署接口
